# Wordle Bot

A simple yet fun wordle discord bot!

# Getting started

## Installation
```bash
git clone https://github.com/qxb3/WordleBot.git
cd WordleBot
```

## Running

> Note: Alternatively You can use npm

```bash
yarn start
yarn dev #Running development
```
